About Crypto Hindsight

This website has been a group collaboration from Group 7 of CIS4301 Spring 2021. We have worked together to build a tool that allows the user to search through a few years worth of historical cryptocurrency data. By connecting the Oracle Database to this application, we are able to run SQL queries to retrieve complex trend information that can be used to make educated decisions about cryptocurrency investments. Multiple aspects of both Ethereum and Bitcoin can be analyzed by the user within the different tabs provided.

Complex Trend Queries

  • Query 1: Daily Percentage Change in Price for a Given Crypto.
  • Query 2: Portfolio Value Calculator Over Time.
  • Query 3: Crypto Performance Compared To Market Over Time.
  • Query 4: Trade Volume Of Crypto Over Time.
  • Query 5: Correlation Between Different Cryptos Over Time.
